# Survival Shooter Prototype (Legacy)

The original 2D top-down survival shooter remains at:

- `scenes/gameplay/gameplay.tscn`
- `scripts/actors/`, `scripts/levels/gameplay_controller.gd`
- `resources/levels/`, `resources/enemies/`, etc.

Launch from the main menu via **Survival Prototype**, or `GameManager.start_new_run()`.

This mode is preserved for reference and future dungeon-style gameplay. It does not share runtime state with the 2.5D life-sim vertical slice.
